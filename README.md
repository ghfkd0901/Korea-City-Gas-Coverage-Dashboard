## 📊 소개

한국도시가스협회의 공식 통계 데이터를 기반으로 전국 도시가스 보급 현황을 시각화한 인터랙티브 대시보드입니다.

### ✨ 주요 기능

- 📈 **시도별/회사별 보급률 추이** 시각화
- 📊 **세대수/수요가수 변화** 추적
- 📉 **전년대비 증감률** 분석
- 🔴 **감소 지점 자동 표시** (빨간 네모)
- ⭐ **무감소 우수 지역/회사** 별표 표시
- 📥 **엑셀 다운로드** 기능 (전체 데이터/필터 데이터)
- 🎨 **강조 표시** 기능으로 특정 시도/회사 비교
